# vue-function-api体験

## TL;DR
だいたいcustom hooksだ  
vueとlogicの分離がすごい簡単  
さよならrender props、scoped slots、HoC、function as children！ 

## 完走した感想
意外にとっつきやすく、scoped slotsにbreaking changeが入ってショックだった自分には嬉しいAPIでした。  
現時点で特殊な表現も少なく可読性も担保されているように感じます。  
設計はかなり変わって来そうですが、vue3.0が楽しみです。
